# Air Pollution Analysis in Astana

##  Project Overview
This project analyzes air pollution in Astana using real environmental data.  
The aim is to identify seasonal trends, pollution sources, and differences between monitoring stations.

##  Data
- Pollutants: PM2.5, PM10, CO, NO2, SO2  
- Time: date, seasons, heating vs non-heating period  
- Multiple monitoring stations across the city  

##  Analysis
- Seasonal comparison (heating vs non-heating)
- Station-based comparison
- Identification of high-risk areas
- Trend analysis of pollutants

##  Key Findings
- Pollution is significantly higher during the heating season  
- PM2.5 and PM10 are the most dangerous pollutants  
- Some stations consistently show higher pollution  
- Construction activity may increase dust and particulate matter  

##  Recommendations
- Switch from coal to gas heating  
- Improve public transport  
- Reduce emissions from vehicles  
- Control construction dust  
- Increase green zones  

##  Tools
Python (Pandas, Matplotlib, Seaborn), Jupyter Notebook
